Import module:
    >>> MyList = __import__('1-my_list').MyList

Note: assume that all the elements of the list will be of type int 

TESTS
test 0: case positive integers
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(-4)
    >>> my_list.append(2)
    >>> my_list.append(-3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()

    >>> my_list = MyList()

    []
